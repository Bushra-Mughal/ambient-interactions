# Ambient interactions
*interactive visual elements for Front-end devs*

A collection of fun, creative and interactive visual elements built with HTML Canvas and JavaScript.

These elements explore:
- Soft motion and fluid interaction
- Eye-friendly visuals
- Cursor-driven dynamics
- Minimal, modern front-end aesthetics

Each visual element is self-contained and runs directly in the browser.

## Elements:
- Cosmos
- Dynamic Lines
- Magnetic Flow Grid
- Aurora Strings
- Breathing Mist
- Breathing Mist (Interactive)
- Soft Box Field
- Living Isometric City

## Usage:
Open any `.html` file directly in the browser.
